# Foody Love Restaurant

A simple, elegant restaurant website showcasing delicious food and excellent service.

## Description

Foody Love is a static HTML website for a fictional restaurant. It features a modern design with sections for the menu, about us, special recipes, photo gallery, and contact information. The site is built with pure HTML and CSS, making it lightweight and easy to deploy.

## Features

- **Responsive Design**: Works on desktop and mobile devices
- **Hero Section**: Eye-catching banner with call-to-action
- **About Section**: Information about the restaurant's quality and service
- **Special Recipes**: Highlighting unique dishes
- **Photo Gallery**: Beautiful food photography
- **Menu**: Organized by categories (Drinks, Main Course, Desserts)
- **Mobile App Promotion**: Links to download the restaurant's app
- **Contact Form**: For customer inquiries
- **Footer**: Contact information and opening hours

## Technologies Used

- HTML5
- CSS3

## How to View

Simply open `index.html` in any modern web browser.

## Assets

The website uses high-quality food images from Unsplash. Local copies of some images are available in the `asset/images/` folder.
